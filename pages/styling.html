Grid

We have 2 types of grid layouts. One is a grid layout for an entire page and the other is a grid container you can use everywhere you want.

Grid for an entire page

This grid layout places <header>, <main>, <aside>, and <footer> automatically. To use this grid layout, you wrap your <header>, <main>, <aside>, and <footer> inside <div> with class = “bodyGrid”.

Option:
By default, <aside> is placed on the right of the page. If you want <aside> to be on the right, add class = “asideL” into <div class = “bodyGrid”>. If you don't have <aside>, add class = “noAside” into <div class = “bodyGrid”>.

Grid container

This container has 1 to 6 columns that change responsively. To use this grid container, put class = “item” into child elements, like <div class = “item”> a grid item, and wrap it with <div class="container">.

Option:
By default, the max width of this grid container is 1000px. To make the width 100%, add class = “width100” into <div class="container">.

Option:
By default, <div class = “item”> is placed in 1 column and 1 row (does not span). To span grid items, add additional classes into <div class = “item”>. See details below.

Utlility Classes

Quick floats and clearfix

Put class = "float-left" or class = "float-right" into the element you want to float. You can use "clearfix" to clear floats. Put class = "clearfix" into the parent element.

                        
<div class="clearFix">
    <img class ="float-right" src="dummy.jpg" quot;dummy image">
    <p>Text</p>
</div>

No print class

Use class = "noPrint" to the element that you don't want users to print.
