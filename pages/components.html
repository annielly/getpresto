    <div id="viewEvent">

Be sure you have properly linked our stylesheet and the calendar.js file for this calendar component. You can also manually add holidays or special events into the holidays variable in the JavaScript file:


    const holidays = [
  {
    hdate: "01-01-2023",
    holiday: "New Year's Day",
  },
  {
    hdate: "06-17-2023",
    holiday: "JAM SESH",
  },
  {
    // To add desired holidays or special events, input like the format below to this array
    // hdate: "25-12-2023",
    // holiday: "Christmas",
  },
];

Parallax Scrolling

You can make a parallax website with our framework. To make a parallax effect, add class="parallaxBg” into <body> tag. You need to name your background image “parallaxImage.jpg” and put the image file into “images” folder. You can check how it looks like on the parallax demo page

Option:
If you put the class=”scrollFadeIn” into the element, the element fades in when users scroll. Make sure to link your HTML to the JavaScript library named “scrollIn.js”.

Option:
If you want to make your background image blur, add class =”parallaxBgBlur” into <body>.

Note: You can’t use “Grid for an entire page” when you use this parallax effect. (You can use “Grid container”.)

Video Embedding:

Make a section with class="videos". In order to embed the videos, you will need the iframe tag with the video's URL, which will be found on the videos's hosted site. Then within the section, you will put the iframe into a div with class="video". Underneath this, you can add your text with h3 and p tags, respectively. Finally, wrap both the div containing the iframe AND the text within another div with class="vid vid1". Note: you will change out class="vid1" for vid2, vid3, or vid4 depending on how many videos you add. For example, the second video would have class="vid vid2"You may add up to 4 videos per page.
