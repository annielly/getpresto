Welcome to the Presto Framework!

We are a website framework that was created with the musical entreprenuer in mind. With Presto!, you can quickly create a personalized website to help promote your work, announce tour dates, and more. We named our framework after the musical tempo "presto" because we get you up and running super fast so you can get back to doing what you love quicker--music!

How to use Presto!

Step 1:Download the Presto! Framework!

Step 2: Set up your HTML

By default, Presto! inludes links to Font Awesome and Google Fonts, as well as links the index page to the necessary CSS and JavaScript pages. This step is to make you aware of this, and to show how to add these links should you need to.

Step 3: Styling with Presto!

Step 3: Go through the Stylings and Components pages and add the necessary code per the instructions. You can find links in the menu at the top of the page.
